## 使用方式

1. 打开文件，全选复制
2. 粘贴到任意 AI 对话框
3. 在后面附上你的具体需求
4. AI 会按照提示词里定义的流程和格式来回复你

每个提示词都是自包含的——AI 读完后就知道自己是什么角色、该按什么结构输出、该问什么问题。
